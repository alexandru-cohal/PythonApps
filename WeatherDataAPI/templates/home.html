Weather Data API

URL format for requesting the temperature for a specific station and a specific date: http://127.0.0.1:5001/api/v1/station/date

Example: http://127.0.0.1:5001/api/v1/50/2017-08-15

URL format for requesting the temperatures for a specific station and all dates: http://127.0.0.1:5001/api/v1/station

Example: http://127.0.0.1:5001/api/v1/50

URL format for requesting the temperatures for a specific station, a specific year and all dates: http://127.0.0.1:5001/api/v1/yearly/station/year

Example: http://127.0.0.1:5001/api/v1/yearly/50/2017
