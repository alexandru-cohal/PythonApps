URL format for a specific station and a specific date: http://127.0.0.1:5001/api/v1/station/date

URL example: http://127.0.0.1:5001/api/v1/10/1988-10-25

URL format for a specific station and all records: http://127.0.0.1:5001/api/v1/station

URL example: http://127.0.0.1:5001/api/v1/10

URL format for a specific station, specific year and all records: http://127.0.0.1:5001/api/v1/yearly/station/year

URL example: http://127.0.0.1:5001/api/v1/yearly/10/1999
